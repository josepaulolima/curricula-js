# listAllValues

Escreva uma função chamada `listAllValues` que retorna um _array_ de todos os valores do objeto de entrada.

Exemplo de entrada:

```javascript
{
  name: 'Krysten'
  age: 33,
  hasPets: false
}
```

Valor devolvido pela função \(saída\):

```javascript
['Krysten', 33, false]
```
